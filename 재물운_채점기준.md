# 사주 재물운 평가 채점 기준서

**버전**: v2.0 (Method C 적용)
**최종 수정**: 2026-04-10

---

## 1. 개요

사주팔자의 재물운을 정량화하여 **30~100점** 범위로 평가합니다.

| 항목 | 값 |
|---|---|
| 최저 점수 | 30점 |
| 평균 점수 | 70점 |
| 만점 | 100점 |
| 점수 분포 | 정규분포 (σ ≈ 15) |

---

## 2. 점수 산출 구조

```
총점 = 표시점수(원점수)
     원점수 = 1단계(교차보정) + 2단계(교차보정) + 3단계 + 4단계 + 5단계 + 교차추가점
```

**원점수 → 표시점수 변환 (지수 커브)**

```
표시점수 = min(100, round(30 + 70 × (1 − e^(−원점수/24))))
```

| 원점수 | 표시점수 | 등급 |
|---|---|---|
| 0 | 30 | 최하위 |
| 5 | 43 | 재물운 매우 약함 |
| 10 | 54 | 재물운 약함 |
| 15 | 63 | 평균 이하 |
| 20 | 70 | 평균 |
| 25 | 75 | 평균 이상 |
| 30 | 79 | 평균 이상 |
| 40 | 86 | 재물운 매우 강함 |
| 50 | 91 | 극상위 |
| 60 | 94 | 극상위 |

## 4. 1단계: 재성 구조 (최대 35점)

재물운의 근본이 되는 재성(편재·정재)의 존재와 구조를 평가합니다.

| 항목 | 조건 | 점수 |
|---|---|---|
| **재성 미투출** | **재성 없음 (지장간 암장 기운)** | **+12** |
| 재성 유무 | 사주에 재성이 존재 | +10 |
| 재성 위치 | 일지 | +8 |
| | 월지 | +6 |
| | 연지 / 시지 | +4 |
| 재성 종류 | 정재 (안정적 수입) | +5 |
| | 편재 (변동성 수입) | +3 |
| 재성 개수 | 2개 이상 | +4 |
| | 1개 | +2 |
| 재성=용신 | 재성이 용신일 때 (신강) | **+5** (v1.1: +8) |
| 재성=기신 | 재성이 기신일 때 (신약) | **−3** (v1.1: −5) |

**설계 근거**:
- **재성 미투출 +12**: 적천수(滴天髓) — "재성이 투출하지 않아도 지장간에 묻혀 있으면 재물의 근(根)이 있다". 대운/세운에서 재성이 오면 발복 가능. 재성 있을 때의 ~40% 수준으로 잠재 기운 반영
- 일지의 재성이 가장 가치 있음 (일간과 가장 가까운 지지)
- 정재는 안정적 재물, 편재는 투기성이므로 정재에 가산점
- **용신 +5 (축소)**: 용신의 가치는 인정하되, 교차 배율(식신생재 ×1.15)과 중복 보너스를 방지. 용신-기신 편차를 8점(일지 위치 점수와 동등)으로 적정화
- **기신 −3 (완화)**: 기신의 해(害)는 용신의 이익의 대칭이 아님. 재성이 기신인 경우 "재물로 인한 심로(心勞)" 수준이며, 재다신약 교차조건과의 이중 패널티 방지

## 5. 2단계: 생조 구조 (최대 25점)

재성을 생(生)하거나 보조하는 구조를 평가합니다.

| 항목 | 조건 | 점수 |
|---|---|---|
| 식신·상관 존재 | 식상이 재성을 생해줌 | +8 |
| 재성 합(合) | 재성이 합에 참여 | +7 |
| 중화 | 신강도 신약도 아닌 균형 상태 | +10 |
| 신강 또는 신약 | 한쪽으로 치우침 | +5 |

**보정 규칙**:
- 신강인데 재성이 없으면 → 2단계 점수 × 0.5 (재성 없이 신강하면 재물과 무관)

**설계 근거**:
- 식상(식신·상관)은 오행 상생 순서상 재성을 생해주는 관계
- 합(合)은 재성이 나에게 묶여있음을 의미
- 중화(균형)가 가장 좋고, 편향은 일부 감점

---

## 6. 3단계: 방해 요소 (최소 −25점)

재물운을 훼손하는 구조적 요인을 감점합니다.

| 항목 | 조건 | 점수 |
|---|---|---|
| 비겁 과다 | 비견·겁재 3개 이상 | −10 |
| | 비견·겁재 2개 | −5 |
| 재성 충(沖) | 재성이 충을 당함 | −8 |
| 편관 위치 | 편관이 월간에 있음 | −7 |
| | 편관이 기타 위치 | −3 |

**설계 근거**:
- 비겁(비견·겁재)은 재성을 극(克)하는 오행 → 재물을 빼앗김
- 충(沖)은 재성의 힘을 흩뜨림
- 편관은 나를 극하는 관계 → 재물 활용 능력을 제한

---

## 7. 4단계: 시기 보정 (최대 15점)

현재 운(대운·세운)에 따른 재물운의 시기적 가감입니다.

| 항목 | 조건 | 점수 |
|---|---|---|
| 대운 재성 | 현재 대운에 재성 존재 | +8 |
| 대운 관살 | 현재 대운에 관살 존재 | −5 |
| 세운 재성 | 올해 세운에 재성 존재 | +5 |
| 세운 관살 | 올해 세운에 관살 존재 | −3 |
| 대운·세운 합 | 대운과 세운이 육합 관계 | +2 |

**주의**: 대운 재성과 대운 관살은 택일 (재성 우선), 세운도 마찬가지

**설계 근거**:
- 대운은 10년 단위 운세 → 영향력이 크므로 가감폭 큼
- 세운은 1년 단위 → 상대적으로 작은 가감
- 대운·세운이 합이면 운의 에너지가 결합되어 소폭 가산

---

## 8. 5단계: 양방향 보정 (신살)

신살은 양면성(긍정/부정)이 있어 가중 평균으로 계산합니다.

| 신살 | 긍정값 | 부정값 | 긍정확률 | 부정확률 | 기대값 |
|---|---|---|---|---|---|
| 공망 | +5 | −5 | 50% | 50% | 0.0 |
| 원진살 | +2 | −6 | 30% | 70% | −3.6 |
| 도화살 | +4 | −2 | 60% | 40% | +1.6 |
| 역마살 | +3 | −3 | 50% | 50% | 0.0 |

**특수 보정**: 역마 + 공망 동시 → 역마 긍정확률 70%로 상향 (기대값 +1.2)

**계산 공식**: `점수 = (긍정값 × 긍정확률) + (부정값 × 부정확률)`

**설계 근거**:
- 공망: 비어있음. 재물이 비면 손실이지만, 집착이 없어 오히려 기회가 오기도
- 원진살: 대부분 부정적 (재물 관계 불화) → 부정확률 70%
- 도화살: 대인관계 매력 → 재물 유입 가능성이 더 큼 → 긍정확률 60%
- 역마살: 이동·변화 → 재물 기회/위험 반반

## 10. 2단계 보정: 신강+재성 없음

| 조건 | 효과 (v2.0) | v1.1 |
|---|---|---|
| 신강이면서 재성 없음 | 2단계 점수 × **0.7** | ×0.5 |

## 12. 점수 분포 설계 의도

### 왜 지수 커브인가?

선형 매핑(원점수 비례)은 원점수가 높을수록 쉽게 만점에 도달합니다. 지수 커브(`1 − e^(−x/30)`)는:

- **하위권 분별력 확보**: 원점수 0~25 구간이 표시 30~70으로 넓게 펼쳐져 차이가 드러남
- **상위권 수렴**: 원점수 50 이상은 표시 87~100으로 압축 → 만점은 극히 드물게
- **자연스러운 정규분포**: 사주의 다양한 조합이 70점 근처에 모이도록 설계

### 왜 최저 30점인가?

재성이 전혀 없고 모든 조건이 불리해도 30점. 사주 재물운은 "0점"이라는 개념이 부적절하며, 누구에게나 일정 수준의 재물 운은 존재한다는 관점입니다.

### 왜 평균 70점인가?

100점 만점에서 평균 50점은 직관적으로 "나쁘다"는 인상을 줍니다. 70점을 평균으로 설정하면:
- 대부분의 사람이 "보통 이상"으로 느끼는 60~80점대에 분포
- 진정한 약점은 50점 이하로 명확히 구분
- 90점 이상의 희소성으로 상위권의 가치 부각

## 부록: 점수 계산 예시

### 예시 A: 신강 + 식신생재격 (v2.0)

```
1단계: 재성유무(10) + 위치·일지(8) + 정재(5) + 1개(2) + 용신(5) = 30
2단계: 식상(8) + 합(7) + 신강(5) = 20
3단계: 0
4단계: 대운재성(8) + 세운재성(5) = 13
5단계: 역마(0) = 0
교차: #1 식신생재격(30×1.15=34.5, +4.5) → #19 신강보정(34.5×1.043=36, +1.5)
원점수: 36 + 20 + 0 + 13 + 0 = 69
표시점수: 30 + 70 × (1 − e^(−69/24)) ≈ 94점
등급: 극상위 재물 구조 (상위 1~3%)
```

### 예시 B: 재다신약 (재성 2개 + 신약, v2.0)

```
1단계: 재성유무(10) + 위치·월지(6) + 편재(3) + 2개(4) + 기신(-3) = 20
2단계: 식상(8) + 신약(5) = 13
3단계: 비겁2개(-5) = -5
4단계: 0
5단계: 원진(-3.6) = -3.6
교차: #5 상관생재(20×1.10=22, +2) → #2 재다신약(22×0.55=12.1, -9.9)
원점수: 12.1 + 13 + (-5) + 0 + (-3.6) = 16.5
표시점수: 30 + 70 × (1 − e^(−17/24)) ≈ 65점
등급: 평균 (상위 36~65%)
```

### 예시 C: 재성 없음 (v2.0)

```
1단계: 재성 미투출 기본(12) = 12
2단계: 식상(8) + 신강(5) = 13 → 신강+재성없음 보정 ×0.7 → 9.1
3단계: 0
4단계: 대운재성(8) = 8
5단계: 도화(+1.6) = 1.6
교차: 없음
원점수: 12 + 9.1 + 0 + 8 + 1.6 = 30.7
표시점수: 30 + 70 × (1 − e^(−31/24)) ≈ 80점
등급: 재물운 강함 (상위 11~20%)
```
